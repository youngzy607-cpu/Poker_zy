# 德州扑克 Web 版 (Texas Hold'em Web)

一个基于 HTML5/CSS3/JavaScript 的单机德州扑克游戏。支持 1-5 名电脑对手，以及 **8人局域网联机对战**。

## 最新更新：非阻塞式买入与旁观模式 (v3.3.1)
本次更新优化了现金桌的破产保护体验，让游戏流程更加流畅。

### 核心功能
1.  **非阻塞式买入 (Non-blocking Rebuy)**
    *   **游戏不暂停**：当您输光筹码时，游戏不再强制暂停。其他电脑玩家会继续进行下一局游戏，您将暂时处于“旁观 (Sitting Out)”状态。
    *   **无缝重返**：在旁观期间，您可以从容地点击弹窗或“+”按钮进行买入。买入成功后，系统会在**下一局发牌时**自动将您加入战局。
2.  **视觉反馈优化**
    *   **旁观状态**：输光筹码后，您的头像会显示橙色的“补充筹码中”呼吸灯提示。
    *   **透明弹窗**：补码弹窗的背景调整为半透明，让您在买入时也能隐约看到牌桌上的战况。
    *   **主动入口**：新增了金色的 **“+”** 按钮，随时都可以点击补充筹码，无需等到破产。

## 历史更新：现金桌破产保护 (v3.3.0)
本次更新为现金桌模式引入了人性化的“破产保护”与“中途带入”机制，提升了游戏的连续性体验。

### 核心功能
1.  **中途带入 (Rebuy)**
    *   **自动触发**：当您在现金桌输光所有筹码时，无需退出房间，系统会自动弹出补码窗口。
    *   **灵活买入**：如果您的账户（Bankroll）中还有余额，可以像开局时一样拖动滑块，自由选择带入金额，立即重返战局。
2.  **破产救济 (Bankruptcy Relief)**
    *   **广告复活**：如果您的账户余额也已耗尽（彻底破产），现在可以通过观看一段 5 秒的模拟广告来获取救济金。
    *   **奖励丰厚**：每次观看广告可获得 **1000 筹码**，助您东山再起。
