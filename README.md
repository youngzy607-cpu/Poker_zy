# 德州扑克 Web 版 (Texas Hold'em Web)

一个基于 HTML5/CSS3/JavaScript 的单机德州扑克游戏。支持 1-5 名电脑对手，具备基础的德州扑克规则逻辑。

## 最新更新：牌型说明功能 (v2.3.0)
本次更新为游戏增加了详细的牌型说明功能，帮助新手玩家快速熟悉德州扑克的大小规则。

### 新增功能
1.  **牌型说明 (Hand Rankings)**
    *   **主菜单入口**：在主界面新增“牌型说明”按钮，点击即可查看完整的牌型大小顺序图文说明。
    *   **游戏内入口**：在单机游戏界面右上角新增“ℹ️”按钮，随时查阅牌型，无需退出游戏。
    *   **图文详解**：涵盖皇家同花顺到高牌的全部10种牌型，并配有直观的卡牌示例。

## 历史更新：成就系统上线 (v2.2.0)
本次更新引入了全新的成就系统，让您的每一次胜利都更有意义！

### 新增功能
1.  **成就系统 (Achievements)**
    *   **多样化挑战**：涵盖“累计胜场”、“特殊牌型胜利”（如第一次同花赢）、“资产积累”等 30+ 项成就。
    *   **丰厚奖励**：每解锁一个成就，都会获得额外的筹码奖励，助您东山再起。
    *   **实时反馈**：游戏内实时弹出成就解锁通知，爽感倍增。
    *   **成就列表**：在主菜单可查看所有成就的进度和解锁状态。

2.  **数据统计升级**
    *   底层数据结构升级，支持记录各种牌型（对子、三条、同花等）的详细获胜次数。

## 历史更新：个人战绩与数据持久化 (v2.1.0)
本次更新为游戏增加了数据存档功能，您的游戏进度将不再丢失。

### 新增功能
1.  **数据持久化**
    *   **筹码保存**：现在，您的筹码数会在本地自动保存。退出游戏或关闭浏览器后，下次进入将继承上次的筹码余额。
    *   **破产保护**：如果您的筹码归零，系统会在下次开局时自动为您补充 1000 筹码。
2.  **个人战绩中心**
    *   **统计概览**：在主菜单点击“个人战绩”，可查看总局数、胜场数、总盈利、单局最大底池等关键数据。
    *   **历史记录**：展示最近 20 局的详细盈亏记录和当时持有的手牌。
    *   **最佳牌型**：系统会自动记录并展示您在历史上拿到过的最强手牌。

## 历史更新：主菜单与导航系统 (v2.0.1)
本次更新为游戏引入了完整的导航流程，优化了用户进入和退出游戏的体验。

### 核心改进
1.  **全新主菜单**
    *   新增启动页，提供“单机模式”、“联机对战（开发中）”、“退出游戏”三个主入口。
    *   将原本直接弹出的单机设置弹窗收纳进“单机模式”子菜单。
2.  **游戏内导航**
    *   **返回大厅**：在游戏界面右上角增加了“主页”按钮，允许玩家中途结束游戏并返回主菜单。
    *   **退出确认**：为关键的退出操作增加了二次确认弹窗，防止误触。

## 历史更新：基于性格驱动的 AI 引擎 (v2.0.0)
引入了具有独立性格特征的 AI 对手，彻底改变了人机对战的体验。
1.  **性格参数化**
    *   **Aggression (侵略性)**: 决定 AI 加注和进攻的频率。
    *   **Tightness (松紧度)**: 决定 AI 入局的手牌范围。
    *   **BluffFrequency (诈唬率)**: 决定 AI 在劣势时偷鸡的概率。
2.  **智能决策**
    *   引入 `OddsCalculator` 实时计算胜率，结合性格参数做出拟人化决策。
    *   AI 决策日志现在会输出到控制台，方便观察其思考过程。

## 历史更新：代码架构重构 (v1.6)
本次更新专注于底层代码的模块化重构，为后续的 UI 焕新计划打下坚实基础。

### 核心改进
1.  **模块化架构**
    *   将原本臃肿的 `script.js` 拆分为职责单一的独立模块：
        *   `js/card.js`: 核心卡牌逻辑。
        *   `js/evaluator.js`: 牌力计算引擎。
        *   `js/player.js`: 玩家状态管理。
        *   `js/ui.js`: 界面渲染与动画控制。
        *   `js/game.js`: 游戏流程主控。
    *   **优势**：逻辑与视图分离，代码可读性与可维护性大幅提升。

2.  **工程清理**
    *   移除了大量冗余的调试代码和废弃注释。
    *   优化了部署文档，去除了环境特定的硬编码路径。

## 最新更新：竖屏重构与部署修复 (v1.5)
本次更新彻底重构了移动端的交互逻辑，并解决了线上部署的兼容性问题。

### 核心改进
1.  **竖屏架构 (Portrait Mode)**
    *   **布局重组**：彻底抛弃了移动端的“缩放桌子”模式，改为上下分层的竖屏布局。
    *   **Action Bar**：底部固定式操作栏，大尺寸按钮，单手拇指即可轻松操作弃牌、过牌、跟注。
    *   **视觉降噪**：移除了移动端不必要的拟物边框，最大化利用屏幕空间显示手牌和公共牌。
2.  **部署修复**
    *   **GitHub Pages 兼容**：新增 `.nojekyll` 文件，修复了因 Jekyll 主题干扰导致的白屏问题。

## 最新更新：移动端适配与身份标识 (v1.4)
本次更新重点优化了移动端体验，并增加了关键的游戏身份标识。

### 新增特性
1.  **移动端深度适配**
    *   **布局重构**：针对手机竖屏（<768px）进行了专门的 CSS 优化，调整为更紧凑的纵向布局。
    *   **触控优化**：增大了操作按钮的尺寸（Min-Height 44px），优化了加注滑块的交互体验，防止误触。
    *   **空间管理**：压缩了底池和公共牌区域的垂直间距，确保在小屏幕上也能完整显示手牌和对手。
2.  **身份可视化标签**
    *   **庄家 (D)**：白底黑字圆形徽章，标记 Dealer 位置。
    *   **小盲 (SB)**：紫底白字徽章，标记 Small Blind。
    *   **大盲 (BB)**：橙底白字徽章，标记 Big Blind。
    *   **动态轮转**：徽章会随着每局游戏 Dealer 的轮换自动更新位置。

## 最新更新：实时牌力提示 (v1.2)
本次更新增加了一个实用的辅助功能，帮助玩家更好地判断当前局势。

### 新增特性
1.  **实时牌力与胜率显示 (v1.3)**
    *   **牌力分析**：实时显示当前手牌的最大牌型（如：高牌 A、对子 4）。
    *   **胜率预测**：基于蒙特卡洛模拟算法，实时计算当前手牌战胜所有对手的概率（例如：胜率 65%）。
    *   **动态颜色**：
        *   <span style="color:#2ecc71">绿色</span>：胜率 > 70%（优势局）
        *   <span style="color:#f1c40f">黄色</span>：30% - 70%（胶着局）
        *   <span style="color:#e74c3c">红色</span>：胜率 < 30%（劣势局）
    *   **布局整合**：信息标签紧贴玩家昵称右侧，清晰直观。

## 最新更新：极速爽感升级 (v1.1)
本次更新专注于提升游戏的视觉反馈和操作流畅度，打造“竞技风”体验。

### 新增特性
1.  **动态发牌动画**
    *   卡牌不再瞬间出现，而是从屏幕中心“飞入”玩家手中。
    *   采用增量渲染技术，确保动画平滑流畅。
2.  **筹码飞行动效**
    *   **下注**：筹码图标从玩家头像飞向底池。
    *   **赢钱**：底池筹码分裂并飞向胜利者。
3.  **视觉反馈增强**
    *   **赢家高亮**：胜利玩家获得金色呼吸灯边框特效。
    *   **弃牌状态**：弃牌玩家头像变灰并半透明，视觉上“退场”。
    *   **当前行动**：醒目的黄色边框提示当前轮到的玩家。
4.  **节奏优化**
    *   缩短了 AI 思考时间和回合间隔，游戏节奏更加紧凑。

## 功能列表
*   **自定义对手数量**：支持 1 到 5 个电脑对手。
*   **标准游戏流程**：Pre-flop, Flop, Turn, River, Showdown。
*   **智能 AI**：基础的电脑决策逻辑（随机 + 简单底池概率）。
*   **牌型判断**：完整的德州扑克牌型比对算法（同花顺 > 四条 > ... > 高牌）。
*   **响应式布局**：适配桌面端浏览器。

## 技术栈
*   HTML5
*   CSS3 (Flexbox, CSS Animations, Transitions)
*   Vanilla JavaScript (ES6+)

## 如何运行
直接在浏览器中打开 `index.html` 即可开始游戏。
